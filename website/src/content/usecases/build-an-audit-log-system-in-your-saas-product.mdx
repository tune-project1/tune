## What this article is about

An audit log keeps a record of user actions in your SaaS app. It helps with security, debugging, and compliance.

For example, you can record when a user updates their billing info or deletes an account.

![Tune in action](https://tune/images/pitch/ops_insitu.jpg)

## What is Tune and how it can be used to track audit events

Tune is an open source toolkit for capturing events in any tech product.

You can log audit events with a simple API call.

It lets you view a timeline of actions and filter by user, event type, or date.

## Setting up Tune

1. Go to [app.tune](https://app.tune) and sign up.
2. Create a new project and copy your API key.
3. Install the SDK in your code base.
