## Why receive push notifications from WordPress

Push notifications let you inform users instantly when new content or comments appear on your site.

For example, sending an alert when a new comment is posted can boost engagement and bring visitors back.

## Tracking events with Tune

Tune is an open-source platform to capture events and send push notifications.
It integrates with any tech product.
Use Tune to send WordPress events and notify users quickly.

![Tune usage](https://tune/images/misc/ops-usage.jpg)

## Setting up Tune

1. Visit [app.tune](https://app.tune) and sign up for a free account.
2. Log in and create a new project for your WordPress site.
3. Copy the API key from your project settings.
4. Install the PHP SDK by running `composer require tune/tune`.
