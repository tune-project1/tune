# Integrations with self-hosted instances

### Nodejs SDK

When using our Nodejs SDK, pass the `baseUrl` as the second parameter when initiating.

<Code lang="js" client:load>
```
import Tune from "@tune/sdk"

const ops = new Tune(`ops_yourapikey`, {
  baseUrl : `https://urlofyourinstancebackend`
});
```
</Code>
